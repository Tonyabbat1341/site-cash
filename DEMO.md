## 🛠️ Architecture technique

### Frontend
- **Next.js 14** avec App Router
- **TypeScript** pour la sécurité des types
- **Tailwind CSS** pour le styling
- **shadcn/ui** pour les composants
- **React Flow** pour les workflows

### Backend
- **Next.js API Routes** pour les endpoints
- **Supabase** pour la base de données
- **Stripe** pour les paiements
- **Webhooks** pour la synchronisation

### Base de données
- **PostgreSQL** via Supabase
- **RLS (Row Level Security)** pour la sécurité
- **Schéma optimisé** pour les performances
- **Triggers** pour l'audit

### IA et APIs
- **OpenAI GPT-4** pour la génération
- **Anthropic Claude** pour l'analyse
- **Google Gemini** pour la diversité
- **Mode Auto** pour l'optimisation

## 🔧 Configuration requise

### Développement local
```bash
# Prérequis
Node.js 18+
npm ou yarn

# Installation
npm install
npm run setup

# Configuration
cp env.example .env.local
# Remplir les variables d'environnement

# Lancement
npm run dev
```
